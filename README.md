# TRABAJO_007
## PYTHON_001 - Prácticas de Python

Este repositorio contiene prácticas básicas de Python para aprender programación.

## Estructura del Proyecto

```
TRABAJO_007/
├── practicas/
│   ├── practica_01_hola_mundo.py     # Primer programa en Python
│   ├── practica_02_variables.py      # Variables y tipos de datos
│   └── practica_03_operaciones.py    # Operaciones matemáticas
└── README.md
```

## Requisitos

- Python 3.6 o superior

## Cómo Ejecutar las Prácticas

### Práctica 01: Hola Mundo
```bash
python practicas/practica_01_hola_mundo.py
```

### Práctica 02: Variables
```bash
python practicas/practica_02_variables.py
```

### Práctica 03: Operaciones Matemáticas
```bash
python practicas/practica_03_operaciones.py
```

## Descripción de las Prácticas

1. **Práctica 01 - Hola Mundo**: Introducción básica a Python con el tradicional "Hola Mundo"
2. **Práctica 02 - Variables**: Aprender sobre variables y diferentes tipos de datos (enteros, decimales, cadenas, booleanos)
3. **Práctica 03 - Operaciones**: Realizar operaciones matemáticas básicas (suma, resta, multiplicación, división, etc.)
